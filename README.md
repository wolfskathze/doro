# Galerie Doro

Private Astro-Website fuer eine Galerie. Enthalten sind Startseite, Dorothea
Spund, Ausstellungen, Events, Kuenstleruebersicht, Kontakt und Impressum.

## Entwicklung

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Die Seite ist fuer GitHub Pages als Projektseite unter `/doro` vorbereitet.

Hinweis: GitHub Pages fuer private Repositories benoetigt einen GitHub-Plan, der
private Pages unterstuetzt. Wenn der Plan das nicht erlaubt, bleibt das Projekt
privat nutzbar, kann aber nicht direkt per GitHub Pages veroeffentlicht werden.
